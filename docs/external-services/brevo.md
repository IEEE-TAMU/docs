# Brevo

We use [Brevo](https://www.brevo.com/) (formerly Sendinblue) as our transactional email provider.

It is used to send automated emails from our applications and services.

## DNS Configuration

We have configured SPF and DKIM records for Brevo to ensure high deliverability.

See [DNS Configuration](../infrastructure/cloudflare/dns.md#transactional-email-brevo) for the specific records.
